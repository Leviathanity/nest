# Web Search Skill

Search the web to find current information, research topics, or verify facts.

## Usage

When the user asks for information that might require up-to-date data or verification:

1. Formulate a clear and specific search query
2. Use the web search tool to find relevant results
3. Evaluate the quality and relevance of sources
4. Summarize the key findings for the user
5. Cite your sources when providing factual information

## Best Practices

- Be specific with search terms - include key context
- Use quotes for exact phrase matching
- Add context like timeframes when recency matters
- Check multiple sources when facts are important
- Prefer authoritative sources (official sites, reputable publications)

## Example Queries

- "latest AI developments 2026"
- "TypeScript 5.0 new features"
- "how to fix ERR_CONNECTION_RESET"
- "best practices for React testing"
- "WebFetch documentation"
