# AutoGLM 微信操作技能

## 概述

使用智谱 AutoGLM 通过自然语言指令控制微信应用。

## 前置条件

1. 设备已连接 ADB（当前设备：192.168.2.144:5555）
2. ADB Keyboard 已安装并启用
3. 智谱 API Key 已配置

## 使用方法

### 直接使用 autoglm 工具

```json
{
  "action": "run",
  "task": "你的指令",
  "deviceId": "192.168.2.144:5555"
}
```

## 发送文件流程

微信发送文件步骤：
1. 进入群聊/私聊
2. 点击底部 "+" 按钮
3. 向左滑动菜单，找到"文件"选项
4. 选择文件（手机/聊天记录/收藏）
5. 点击发送

**注意：** 文件需预先保存在手机本地（手机文件/聊天文件/收藏）

### 示例指令

| 操作 | 指令示例 |
|------|----------|
| 打开微信 | "打开微信" |
| 搜索联系人 | "搜索联系人：邱智坚" |
| 进入群聊 | "进入软件开发群" |
| 发送消息 | "在软件开发群里发送消息：你好" |
| 发送文件 | "在软件开发群里发送文件测试.txt" |
| 查看聊天记录 | "查看与邱智坚的聊天记录" |
| 查看群成员 | "查看软件开发群的群成员" |

## 注意事项

- 发送消息后需用户确认
- 敏感操作（添加好友、支付）需人工确认
- 保持 API Key 有效
