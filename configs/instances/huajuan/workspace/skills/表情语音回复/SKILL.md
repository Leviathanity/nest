# 泰迪熊 TTS 语音视频技能

## 概述

结合 MiniMax TTS 语音合成与泰迪熊表情图片，生成带语音的视频回复。
适合微信等无法直接发送语音的场景。

## 泰迪表情图库（吉卜力风格）

| 编号 | 表情 | 背景场景 | 适用场景 |
|------|------|---------|---------|
| 01 | 😊 开心 | 阳光公园 | 问候、积极回复、感谢 |
| 02 | 🤔 思考 | 温馨图书馆 | 分析问题、解释说明 |
| 03 | 😲 惊讶 | 星空魔法 | 意外消息、惊叹 |
| 04 | 😴 犯困 | 月光卧室 | 无聊、休息相关 |
| 05 | 🙌 兴奋 | 节日庆典 | 庆祝、好消息 |
| 06 | 😳 害羞 | 樱花庭院 | 尴尬、不好意思 |
| 07 | 😎 得意 | 聚光灯舞台 | 骄傲、炫耀 |
| 08 | 🍖 饥饿 | 美食厨房 | 吃饭相关话题 |
| 09 | 😎 耍酷 | 复古城市街景 | 耍帅、装酷 |
| 10 | 💗 爱心 | 日落海边 | 喜欢、表达感情 |

## 使用方式

### 基本用法

```bash
cd ~/.openclaw/workspace/skills/teddy-tts-video/scripts
./teddy-tts.sh "要说的文本" [表情编号] [输出文件]
```

### 参数说明

| 参数 | 必填 | 说明 |
|-----|-----|-----|
| text | ✅ | 要说的文本 |
| emotion | ❌ | 表情编号 01-10，默认自动选择 |
| output | ❌ | 输出 MP4 路径 |

### 自动选择表情

根据关键词自动匹配：
- 开心词：你好、谢谢、高兴、开心 → 01
- 思考词：但是、所以、因为、思考 → 02
- 惊讶词：竟然、居然、没想到 → 03
- 困倦词：累了、休息、睡觉 → 04
- 兴奋词：太棒了、恭喜、庆祝 → 05
- 害羞词：不好意思、尴尬 → 06
- 得意词：我很厉害、骄傲 → 07
- 饥饿词：饿了、吃饭、美食 → 08
- 酷词：帅气、酷 → 09
- 爱心词：喜欢、爱、爱你 → 10

## 示例

```bash
# 自动选择表情
./teddy-tts.sh "你好呀，很高兴认识你！"

# 指定表情
./teddy-tts.sh "太棒了！我们成功了！" 05

# 指定输出路径
./teddy-tts.sh "让我想想..." 02 "/tmp/my_video.mp4"
```

## 工作流程

1. 🔍 分析文本，匹配最佳表情
2. 🎙️ MiniMax TTS 生成语音
3. 🖼️ 选择泰迪熊表情图
4. 🎬 ffmpeg 合并为视频
5. 📱 发送视频到微信

## 技术细节

### ffmpeg 命令

```bash
# 1. 缩放表情图匹配视频尺寸
ffmpeg -i emoji.png -vf scale=320:240 -shortest output.png

# 2. 生成视频并合并音频
ffmpeg -loop 1 -i image.png -i audio.mp3 \
       -c:v libx264 -preset ultrafast \
       -c:a aac \
       -shortest output.mp4
```

### 视频规格

- 分辨率：320x240
- 格式：H.264 + AAC
- 时长：匹配语音长度
