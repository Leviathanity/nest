# Feishu Plugin

This plugin integrates OpenClaw with Feishu (飞书) messaging platform.

## Features

- Receive messages from Feishu users
- Send responses back to Feishu
- Support for text messages
- QR code based pairing

## Configuration

Required configuration:
- appId: Feishu application ID
- appSecret: Feishu application Secret
- verificationToken: Verification token for webhook validation

## Installation

This plugin is automatically loaded when included in the instance configuration.
